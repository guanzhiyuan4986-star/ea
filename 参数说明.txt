一、策略核心思路
─────────────────────────────────────────────────────────────
金貔貅是一款专为XAUUSD（黄金）设计的马丁格尔网格EA。

核心理念：用EMA+SMC六维智能资金概念判断趋势方向，顺势加仓拉均价，
通过ATR动态间距适应不同波动环境，靠篮子止盈统一离场。

关键设计取舍：
- 不预测顶底，顺势追、逆势等
- 盘整行情紧凑加仓快速止盈，单边行情自动拉大间距避免逆势密集加仓
- 对冲作为最后防线，总浮亏超阈值才启动

二、入场信号系统
─────────────────────────────────────────────────────────────
模式：综合评分(COMBINED)

                   满分60 = EMA(30分) + SMC归一化(30分)
                            │              │
            ┌───────────────┘              └───────────────┐
            │                                              │
    EMA(13) vs EMA(34)              6个SMC组件 归一化到0-30分
    金叉/死叉 + 收盘价突破           满分70 → 除以70 × 30
    ↓                                              ↓
    0分 或 30分（只有两种）         0-30分连续值
            │                                              │
            └────────────────┬─────────────────────────────┘
                             ↓
                       综合分 0-60
                       阈值 ≥35 开仓

▸ EMA信号（M5周期，占30分）
  快线EMA(13)上穿慢线EMA(34) 且 M5收盘价在快线上方 → 做多30分
  快线下穿慢线 且 收盘价在快线下方 → 做空30分
  不满足则为0分 → EMA无信号时靠SMC无法单独触发

▸ SMC六维分析（3层周期 × 6组件 = 全景机构行为识别）

  大周期4H —— 大方向
  ┌──────────┬────┬──────────────────────────────────┐
  │ 失衡     │ 15 │ 大实体K线一口吃掉前一根，主力进场   │
  │ 供需区   │ 15 │ 脉冲K前的盘整区，历史密集成交位置   │
  └──────────┴────┴──────────────────────────────────┘

  中周期1H —— 入场时机
  ┌──────────┬────┬──────────────────────────────────┐
  │ 订单块   │ 13 │ 连续2根同向脉冲K前的那根反向K区域  │
  │ 公允缺口 │ 12 │ 3K间跳空未回补，价格倾向返回填坑   │
  └──────────┴────┴──────────────────────────────────┘

  小周期M15 —— 精确点位
  ┌──────────┬────┬──────────────────────────────────┐
  │ 流动空白 │ 8  │ 大实体+几乎无影线，单向扫货痕迹    │
  │ 破坏块   │ 7  │ 突破原订单块区域后翻转，确认反转   │
  └──────────┴────┴──────────────────────────────────┘

  归一化：SMC原始分(0-70) / 70 × 30 = 0-30分

▸ H4趋势过滤（默认开启，2K确认模式）
  做多信号需连续2根H4收盘价在EMA上方
  做空信号需连续2根H4收盘价在EMA下方
  防止逆大势开仓

▸ CCI极端过滤（可选开启，H1周期14，阈值±150）
  做多时CCI>150 → 否决（超买不追）
  做空时CCI<-150 → 否决（超卖不追）

三、马丁加仓机制
─────────────────────────────────────────────────────────────
▸ 加仓参数
  最大层数     30层
  首单手数     0.01
  手数倍率     每层×1.20
  单层上限     1.50手
  总手数上限   2.00手
  冷却时间     平仓后45秒

▸ ATR动态间距（核心创新）

  间距 = 100 + 层数×20 + ATR(3)/Point × 系数 × ratio^1.5

  两个ATR句柄（均在M5周期）：
    ATR(3) = 最近3根M5的平均波幅 — 捕捉短期波动
    ATR(6) = 最近6根M5的平均波幅 — 作为中期基准

  扩张比 ratio = ATR(3) / ATR(6)
    ratio=1.0 → 盘整，expansion=1.0，无额外放大
    ratio>1.0 → 波动爆发，expansion>1.0，ATR贡献被放大
    ratio^1.5曲线：平时趴着不动，波动爆时跳起来

  设计意图：
    震荡时ATR小且ratio≈1.0，间距紧凑频繁加仓快止盈
    单边时ATR涨且ratio>1.0，间距自动拉开避免逆势密集追

  实际举例（系数0.05，23层）：
    盘整 ATR=$2.0 ratio=1.0 → 660点 / $0.66
    波动 ATR=$5.0 ratio=1.5 → 1238点 / $1.24
    暴走 ATR=$7.0 ratio=1.7 → 1520点 / $1.52

四、止盈止损体系
─────────────────────────────────────────────────────────────
▸ 篮子动态止盈
  TP = 8美分 + (层数-1)×8美分
  层数越多目标越高、覆盖加仓成本

▸ 动态追踪止盈
  浮盈保留峰值×50%
  启动门槛 = 当前TP×60%
  例：TP=24美分 → 浮盈达14.4美分启动 → 回撤到峰值的50%平仓
  注意：对冲激活期间追踪暂定，由对冲释放接管

▸ 硬止损（默认不限）
  整篮子亏损达设定值全平

▸ 快速亏损熔断（默认关闭）
  N秒内亏损达X美分 → 临时停单 → 回本Y美分恢复

五、对冲保护
─────────────────────────────────────────────────────────────
▸ 触发条件（二选一）
  百分比模式：浮亏占权益≥40%
  绝对金额模式：浮亏≥200美分

▸ 对冲方式
  开启反向仓位 = 马丁总手数×60%
  对冲期间所有风控判断（硬止损/日亏/熔断/追踪）改用
  马丁+对冲的总浮亏，避免只算马丁单边而误杀

▸ 释放方式
  固定止盈：总浮亏回本+额外N美分
  动态止盈：每层N美分递增
