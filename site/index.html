FEATURES

核心优势

为什么选择金貔貅？

🎯

多周期SMC协同

4H看方向→1H找入场→15M精确定位，三个时间周期各司其职，避免单一周期误判

📐

ATR自适应间距

不再使用固定点数加仓，间距随市场波动自动调整。波动大→间距宽，波动小→间距紧，更合理

🔬

七大SMC信号可选

Imbalance、Supply/Demand、Order Block、FVG、Liquidity Void、Breaker + EMA，每个都可独立开关

⚖️

综合评分体系

每种信号分配权重，综合打分过阈值才入场。不是"一个信号就干"，而是"多个信号确认才动手"

🛡️

五重风控保护

日亏损锁仓 + 硬止损强平 + 移动止盈回撤 + 快速亏损熔断 + 亏损对冲保护，层层设防

⏱️

智能时段管理

北京时间统一管理，自动检测服务器时区，支持定时暂停交易（避数据行情），交易时段可自定义
